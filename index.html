Conectar Supabase

Para sincronizar entre dispositivos y pestañas en tiempo real.

Setup (una sola vez)

  1. En tu proyecto de Supabase: SQL Editor → New query. Pegá y corré esto: 
    create table if not exists pragma_state (
  id text primary key,
  data jsonb not null,
  writer text,
  updated_at timestamptz default now()
);

alter table pragma_state enable row level security;

create policy "open access" on pragma_state
  for all using (true) with check (true);

alter publication supabase_realtime add table pragma_state;
  2. Andá a Project Settings → API. Copiá Project URL y la anon public key.
  3. Pegalas abajo.
